# Freebie

A simple freelance deal tracker to simplify and streamline freelancer-client collaboration.

## Introduction
Freelancers often struggle with managing deals and maintaining seamless communication with clients. **Freebie** is here to help! With Freebie, freelancers can easily create deals, share them with clients, and track progress on shared dashboards.

## Features
- **Create and Manage Deals**: Easily create deal entries with all relevant details.
- **Shareable Links**: Send deal links to clients for confirmation.
- **Track Progress**: Both freelancers and clients can monitor the progress in real-time.
- **Secure and Simple**: Built with HTML, CSS, JavaScript, and Firebase for lightweight and reliable performance.

## How It Works
1. **Freelancer creates a new deal**: Add details such as project name, deliverables, deadlines, and agreed costs.
2. **Share confirmation link**: A unique link is generated and sent to the client.
3. **Client confirms the deal**: The client approves the deal and both parties can start tracking.
4. **Real-time tracking**: Both users can track progress and updates via shared dashboards.

## Getting Started
Follow these steps to set up and run the project locally:

### Prerequisites
- Ensure you have a basic web server environment (e.g., XAMPP, Node.js, etc.).
- Set up Firebase for backend functionality.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mobilitysync/freebie.git
   cd freebie
   ```
2. Set up Firebase:
   - Create a Firebase project.
   - Add your Firebase configuration to the app.
3. Run the app:
   - Use a local server to serve files or deploy online.
